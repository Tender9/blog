**函数参数-arguments-对象**

```js
    function fn() {
        arguments.__proto__ = Array.prototype;
        return arguments.reduce((pre, cur) => {
            return pre + cur;
        });
    }

    fn(19, 27, 4, 56, 6, 4, 32, 76, 32, 11, 4, 13, 22, 32);
```



`arguments` 是类数组对象，具有`length`属性，不具有数组方法
